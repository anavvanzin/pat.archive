# MeloVanzin ♡

> *um universo só nosso 🫧*

Uma mini-app interativa — carta de amor disfarçada de jogo retrô — feita por Ana para Lucas.

## Como rodar

```bash
cd melovanzin
npm install
npm run dev
```

Abra `http://localhost:5173` no browser.

## Stack

- **Vite + React + TypeScript**
- **Tailwind CSS** — dark theme com paleta roxa/pink
- **Framer Motion** — transições entre mundos
- **Zustand** — estado global (inventário, beats salvos, scores)
- **Press Start 2P** — fonte pixel art
- **Canvas API** — pixel art procedural (Ana, Lucas, Capivara)

## Mundos

| # | Mundo | Descrição |
|---|-------|-----------|
| 🎵 | **Fruit Loops** | Step sequencer FL Studio — faça beats, salve no localStorage |
| ⚔ | **Tibia** | RPG idle com baú de mensagens, capivara NPC, itens de loot |
| 🏆 | **Bot Lane** | Lane game LoL — acerte minions, coloque wards, abata o dragão |
| 💬 | **Discord** | Chat interativo com a história de origem de Ana & Lucas |

## Easter Eggs

1. **Fruit Loops**: kick nos passos 0,4,8,12 + snare nos passos 4,12 → *"esse era o beat favorito dele 🎵"*
2. **Tibia**: clique na capivara → *"🐾 vamo si ama!!"*
3. **Tibia**: abra o baú 5 vezes para ver todas as mensagens secretas
4. **Bot Lane**: abata o dragão → heart burst + *"DRAGON SLAIN ♡"*
5. **Discord**: leia #primeiro-contato até o fim para ver como tudo começou
6. **✦ button** (canto inferior esquerdo): 10 mensagens de amor para ciclar
7. **Notificações**: aparecem aleatoriamente a cada ~10-20s
