# Memory

## Sobre este painel
Painel de produtividade da **Patricia**. A usuária é a própria Patricia; Ana ajuda a montar e organizar.
Tudo em português.

## Áreas (seções de tarefa)
| Área | O que entra |
|------|-------------|
| **Caça-apartamento** | Visitas, contatos de anúncio, comparação de apês (Florianópolis, aluguel em R$) |
| **Orçamento / finanças** | Contas, receitas, metas de gasto do mês |
| **Sonhos e metas** | Desejos de curto, médio e longo prazo |
| **DJ sets / música** | Ideias de set (techno, tech house), tracklists, datas de tocar |
| **Compromissos / agenda** | Datas, prazos, lembretes |
| **Dia-a-dia** | Organização de tarefas soltas; interesse em organizar invoices/arquivos |

## Ferramentas relacionadas
- `../planejamento-vida.html` — hub visual da Patricia (calendário, orçamento, caça-apê, sonhos, DJ sets)
- `dashboard.html` — este painel de tarefas + memória
