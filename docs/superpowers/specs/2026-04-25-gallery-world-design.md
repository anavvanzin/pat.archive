# Galeria dos Sentimentos (Gallery World) Design Spec

## 1. Overview
The "Galeria dos Sentimentos" is an immersive world within the MeloVanzin ecosystem designed for students to engage with the Iconocracy corpus. It is not merely an archive, but a reflective journey where each art piece acts as a node for deep visual interpretation and auditory discovery.

## 2. Architecture & Components
- **World ID**: `gallery`
- **Component**: `GalleryScreen.tsx` (New)
- **Data Source**: Gallica MCP Server (via `gallica-mcp-server`)
- **Interaction Logic**:
  - **Navigation**: "Gallery Walk" (smooth camera pans between pieces).
  - **Interaction**: Clicking on an artwork reveals a "Lente de Aumento" (Magnifying Glass) modal.
  - **Content**: Visual art, OCR metadata from Gallica, and the audio analysis transcribed via Whisper.

## 3. UX Design ("Fofo" Aesthetic)
- **Visuals**: A soft, dreamlike palette (lavender, soft pink, deep indigo).
- **Interactions**: Subtle particle effects ("heart-bursts" from the audio overhaul) when a student "likes" or saves an interpretation.
- **Audio**: A bespoke ambient loop for the gallery, distinct from the upbeat game tracks, fostering a focus-mode environment.

## 4. Technical Strategy
- **Asset Loading**: Use the existing `gallica-mcp-server` to fetch imagery on-demand.
- **Transcriptions**: Each art piece will be linked to a transcription generated by `Whisper`.
- **Persistence**: "Pinned" items are added to the global `Zustand` store, persisting student collections.

## 5. Success Criteria
- [ ] Students can navigate through a curated gallery walk.
- [ ] Clicking an art piece displays the metadata and audio transcript.
- [ ] Students can "save" pieces to their personal `localStorage` collections.
- [ ] Visuals maintain the "retro-fofo" aesthetic of the game.
