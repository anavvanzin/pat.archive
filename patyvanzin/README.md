Uma mini-app interativa — carta de amor disfarçada de jogo retrô — feita de Ana para Patrícia.

## Como rodar

```bash
cd patyvanzin
npm install
npm run dev
```

Abra `http://localhost:5173` no browser.

## Stack

- **Vite + React + TypeScript**
- **Tailwind CSS** — dark theme com paleta roxa/pink
- **Framer Motion** — transições entre mundos
- **Zustand** — estado global
- **Press Start 2P** — fonte pixel art
- **Canvas API** — pixel art procedural (Ana, Patrícia)

## Mundos

| # | Mundo | Descrição |
|---|-------|-----------|
| 🎹 | **Studio** | Sequenciador FL Studio & DJ Mixer — crie beats e mixe sets em tempo real |
| 🖼️ | **Memórias** | Galeria de fotos Polaroid com as fotos favoritas da Paty |
| 🍳 | **Receitas** | Livro de receitas fofas com ingredientes secretos |

## Easter Eggs

1. **Hub**: cliques rápidos nos utensílios de talher escondidos no rodapé do Hub
2. **Receitas**: clique no ingrediente secreto "amor/carinho" nas receitas
3. **Memórias**: veja e clique em todas as 5 fotos Polaroid
